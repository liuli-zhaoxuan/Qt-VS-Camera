# Qt-VS-Camera
Camera image acquisition software based on Qt+VS.
文件说明：
1. 1.Enhance 图像增强的代码
2. 2.Filter  图像滤波变换代码
3. 3.Qt_Camera  相机采图程序

## 说明：
前两个工程代码是我自己练习 QT + VS 的练手项目，是在B站看视频（在Qt Creator开发），然后自己学着在VS中实现。

第三个是结合高德红外相机SDK、OpenCV 的图像采集简单程序。功能比较简单，作为项目练手。
